# 🛍️ React + TypeScript + Vite Product Catalog

This project is a React + TypeScript + Vite application showcasing a simple product catalog with:

- Modular components (`ProductCard`, `ProductModal`)
- External CSS styling (`components.css`)
- Dummy product data with images, short descriptions, and detailed views
- Interactive modal pop‑ups for product details
- Responsive and maintainable project structure

## 🚀 Features

- React + Vite: Fast dev server with HMR (Hot Module Replacement).
- TypeScript: Strong typing for models and props.
- Modular Components: Product cards and modal separated for clarity.
- Styling: Centralized `components.css` for consistent UI.
- Dummy Data: Products with short and long descriptions, prices, categories, and images.
- Interactive UI: Hover effects on cards, modal overlay for details.

## 🛠️ Setup

1. Install dependencies:
  ```bash
  npm install
  ```
2. Run dev server:

  ```bash
  npm run dev
  ```

Open http://localhost:5173 in your browser.
